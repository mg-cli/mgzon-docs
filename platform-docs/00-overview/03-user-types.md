# User Types

## Overview
The platform supports five primary user types, each with distinct roles, permissions, and capabilities.

## User Type Matrix

| Type | Primary Function | Key Capabilities | Dashboard | Monetization |
|------|------------------|------------------|-----------|--------------|
| **Admin** | Platform management | Full system control, user management, settings | Admin Panel | Platform fees |
| **Seller** | Sell products | Store management, product catalog, orders, inventory | Seller Dashboard | Product sales |
| **Developer** | Sell apps/integrations | App publishing, API management, revenue tracking | Developer Dashboard | App commissions |
| **Client** | Hire freelancers | Project posting, contract management | Client Portal | Service payments |
| **Employee** | Internal work | Task management, salary, HR | Employee Portal | Salary |

## Admin

### Description
Platform administrators with full system access for managing users, content, settings, and operations.

### Permissions
- User management (create, suspend, delete)
- Role and permission assignment
- Platform settings configuration
- Content moderation
- Financial oversight
- Report generation
- System health monitoring
- Audit log access

## Seller

### Description
Users who create and manage online stores to sell products to customers.

### Permissions
- Create and manage stores
- Publish and manage products
- Process orders
- Manage inventory
- Configure store design
- View sales analytics
- Request withdrawals
- Manage payment methods

## Developer

### Description
Users who create and publish apps, integrations, and themes for the platform marketplace.

### Permissions
- Create and publish apps
- Manage API keys
- View installation analytics
- Track revenue and commissions
- Manage referrals
- View tier progress
- Request withdrawals

### Tier System
| Tier | Minimum Earnings | Commission Discount | Benefits |
|------|------------------|---------------------|----------|
| Bronze | $0 | 0% | Basic support |
| Silver | $1,000 | 5% | Priority support |
| Gold | $5,000 | 10% | Featured placement |
| Platinum | $20,000 | 15% | Dedicated account manager |

## Client

### Description
Users who hire freelancers for projects and services.

### Permissions
- Post projects
- Review and accept bids
- Manage contracts
- Release milestone payments
- Submit disputes
- Leave reviews

## Employee

### Description
Internal staff members who work for the platform.

### Permissions
- Role-based permissions
- Task management
- Time tracking
- Salary information access
- Internal communication

## Account Statuses

| Status | Description |
|--------|-------------|
| **Active** | Full access to platform features |
| **Pending** | Awaiting verification or approval |
| **Suspended** | Temporarily restricted |
| **Banned** | Permanently blocked |
| **Inactive** | Account disabled by user |

## Verification Levels

| Level | Requirements | Limits |
|-------|--------------|--------|
| **Basic** | Email confirmation | Daily withdrawal: $500 |
| **Verified** | ID verification | Daily withdrawal: $5,000 |
| **Premium** | Full KYC + business docs | Daily withdrawal: $50,000 |
